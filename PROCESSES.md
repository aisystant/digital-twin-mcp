# Процессы MCP-сервиса цифрового двойника

> Внутренние процессы digital-twin-mcp.
> Межсистемные сценарии → `ecosystem-development/PROCESSES.md`.

---

## 1. Обработка MCP-запроса

> Тип: внутренний процесс
> Владелец: MCP-сервис

**Вход:** MCP-запрос от Claude Code (tool call)

**Действие:**
1. Парсинг запроса (tool name + parameters)
2. Маршрутизация к соответствующему обработчику
3. Запрос данных (БД / файлы / внешние API)
4. Формирование ответа

**Выход:** MCP-ответ (JSON) → Claude Code

---

## 2. Поиск по знаниям

> Тип: внутренний процесс
> Владелец: MCP-сервис (knowledge tool)

**Вход:** Поисковый запрос (текст)

**Действие:**
1. Векторизация запроса
2. Поиск в индексе знаний
3. Ранжирование результатов
4. Форматирование ответа

**Выход:** Релевантные фрагменты знаний с источниками
