# Digital Twin MCP — MAPSTRATEGIC — Стратегическая карта

> Стратегическое видение: версии, фазы, горизонт от месяцев до нескольких лет.
> **Операционные планы** (РП на день/неделю/месяц) — см. [WORKPLAN.md](WORKPLAN.md).

## Классификация показателей (IND.1–4)

```
ПЕРВИЧНЫЕ ДАННЫЕ:
├── IND.1.* Декларируемые — пользователь вводит сам
│   └── Профиль, цели, самооценка, предпочтения
│   └── ✅ Пользователь может редактировать
│
└── IND.2.* Собираемые — автоматически из действий
    └── Курсы, время, платежи, активность
    └── 🔒 Только просмотр

ВТОРИЧНЫЕ ДАННЫЕ:
├── IND.3.* Расчётные — вычисляются, хранятся
│   └── Агентность, ступень, мастерство, риски
│   └── 🔒 Запрет на изменение пользователем
│
└── IND.4.* Генерируемые — по запросу, не хранятся
    └── Рекомендации, прогнозы, сравнения
    └── ⚡ Создаются на лету
```

## Матрица прав доступа

| Тип | Пользователь | Проводник (ИИ) | Система |
|-----|--------------|----------------|---------|
| IND.1.* | Read/Write | Read | Read/Write |
| IND.2.* | Read | Read | Write |
| IND.3.* | Read | Read | Write (calc) |
| IND.4.* | Read | Read/Generate | Generate |

## v1.0 — Декларируемые (IND.1.*)

**Статус:** 🟢 Реализовано

### Функциональность
- Базовый профиль пользователя
- Цели и намерения обучения
- Самооценка текущего уровня
- Предпочтения по формату и расписанию

### Показатели (20 штук)
- **1_1_profile** — Профиль и идентификация
- **1_2_goals** — Цели обучения, проекты, результаты
- **1_3_selfeval** — Выбранные руководства, роли, методы
- **1_4_context** — Расписание, эмоции, контекст

### API
```
write_digital_twin(path="1_declarative/...", data=...)  ✅ Разрешено
read_digital_twin(path="1_declarative/...")             ✅ Разрешено
```

## v2.0 — Собираемые (IND.2.*)

**Статус:** 🟡 Частично (схема готова, интеграции нет)

### Функциональность
- Интеграция с LMS (курсы, модули, тесты)
- Интеграция с ботами (сессии с Проводником)
- Трекинг времени (помодоро, длительность)
- Финансовые данные (подписки, платежи)
- Активность в клубе (посты, комментарии)

### Показатели (планируется)
- **2_1_account** — Сессии, входы, устройства
- **2_2_courses** — Прохождение курсов и модулей
- **2_3_practice** — Выполненные задания
- **2_4_time** — Время сессий, помодоро
- **2_5_community** — Активность в клубе (1 показатель)
- **2_6_coding** — Рабочее время и кодирование / WakaTime (7 показателей, WP-106)
- **2_7_iwe** — Платформенная активность IWE (14 показателей, WP-106)
- **2_8_ai_logs** — Логи общения с ИИ (1 показатель)
- **2_9_finance** — Подписка, платежи (3 показателя)

### Требуется
- [ ] Интеграция с Aisystant LMS API
- [ ] Интеграция с Telegram-ботом
- [ ] Webhook для платёжной системы
- [ ] API клуба (Aisystant Club)

### API
```

read_digital_twin(path="2_collected/...")               ✅ Разрешено
```
