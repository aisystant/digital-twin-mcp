# IND.3.8.01

**Name:** Степень квалификации
**Name (EN):** Qualification degree
**Type:** categorical
**Format:** structured_object

## Description

Профессиональная аттестация методсовета МИМ. Читается из LMS — платформа не вычисляет самостоятельно.

**Три ключевых свойства:**
1. **Только растёт** — степень никогда не понижается
2. **Назначается вручную** — методсовет МИМ, не автоматика
3. **Сквозная** — не привязана к конкретной роли Созидателя, оценивает человека целиком

**Степень ≠ Ступень.** Ступень — автоматически измеряемое мастерство внутри конкретной роли. Степень — профессиональная аттестация.

**Степень ≠ Тир.** Тир — конфигурация доступа к данным платформы. Степень — независимая характеристика развития.

На основе степени начисляются баллы с мультипликатором в экономике платформы (DP.ECON.001).

## Допустимые значения

`DEG.Freshman` | `DEG.Student` | `DEG.Worker` | `DEG.Strategist` | `DEG.Specialist` | `DEG.Practitioner` | `DEG.Master` | `DEG.Reformer` | `DEG.PublicFigure`

Полное описание каждой степени: `metamodel/_shared/degrees.md`

## Формат хранения

```json
{
  "current": "DEG.Student",
  "history": [
    {
      "degree": "DEG.Student",
      "assigned_at": "2025-11-01",
      "assigned_by": "Методсовет МИМ",
      "basis": "Подтверждён ежедневный слот > 90 дней, WakaTime 12ч/нед",
      "expires_at": "2028-11-01"
    }
  ]
}
```

**Обязательные поля при записи:** `degree`, `assigned_at`, `assigned_by`, `basis`, `expires_at`
