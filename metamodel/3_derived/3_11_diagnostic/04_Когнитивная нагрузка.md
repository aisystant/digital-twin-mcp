# IND.3.11.04

**Name:** Когнитивная нагрузка
**Name (EN):** Cognitive load
**Type:** scale
**Format:** float
**Unit:** 0-1

## Flags
- for_prompts: true
- for_qualification: true
- trainee_model: false

## Description

Оценка текущей когнитивной нагрузки: насколько сложен материал относительно способностей ученика.

- **0.0--0.3** -- материал слишком прост (скука, нет роста)
- **0.3--0.7** -- оптимальная зона (Zone of Proximal Development)
- **0.7--1.0** -- перегрузка (фрустрация, бросание задач)

Вычисляется из: Bloom-уровень текущих задач vs результаты (pass_rate, время ответа, количество попыток). При перегрузке система рекомендует снизить сложность (IND.1.5.05) или темп (IND.1.5.04).
