# IND.3.11.03

**Name:** Рекомендуемый фокус
**Name (EN):** Recommended focus
**Type:** semantic
**Format:** structured_text

## Flags
- for_prompts: true
- for_qualification: false
- trainee_model: true

## Description

Области с наибольшим GAP между текущим и целевым уровнем. Структура:

```json
[
  {
    "area": "string",
    "current_mastery": 0.0-1.0,
    "target_mastery": 0.0-1.0,
    "gap": 0.0-1.0,
    "priority": "high|medium|low"
  }
]
```

Вычисляется из: IND.3.5 (mastery по ролям) + IND.3.3.04 (worldview_gaps из BKT) + IND.3.4 (ступень).
Приоритизация: PD.FORM.029 (агентность > мастерство > ресурсность > стабильность > баланс).
