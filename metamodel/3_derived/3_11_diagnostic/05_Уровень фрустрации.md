# IND.3.11.05

**Name:** Уровень фрустрации
**Name (EN):** Frustration level
**Type:** scale
**Format:** float
**Unit:** 0-1

## Flags
- for_prompts: true
- for_qualification: true
- trainee_model: false

## Description

Индикатор эмоционального барьера в обучении. Высокая фрустрация = риск отказа от обучения.

- **0.0--0.2** -- низкая (комфортно, но может быть скучно)
- **0.2--0.5** -- умеренная (продуктивное напряжение)
- **0.5--0.8** -- высокая (снижает мотивацию)
- **0.8--1.0** -- критическая (бросание задач, уход)

Вычисляется из поведенческих сигналов:
- Бросание задач (начал, не завершил)
- Снижение сложности в настройках
- Падение регулярности после сложных тем
- Резкое сокращение времени сессий

При > 0.5: Nudge Engine предлагает поддержку, Портной переключает на более простой материал.
