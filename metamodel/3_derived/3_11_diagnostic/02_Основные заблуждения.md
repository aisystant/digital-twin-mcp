# IND.3.11.02

**Name:** Основные заблуждения
**Name (EN):** Key misconceptions
**Type:** semantic
**Format:** structured_text

## Flags
- for_prompts: true
- for_qualification: true
- trainee_model: false

## Description

Список выявленных заблуждений (misconceptions) с уровнем уверенности. Структура:

```json
[
  {
    "misconception_type": "string",
    "area": "string",
    "confidence": 0.0-1.0,
    "source": "test|chat|practice",
    "detected_at": "ISO date"
  }
]
```

Источник: DP.ARCH.003 SS5.4 (Misconception Map). Экстрагируется асинхронно через LLM из тестов и чатов.
Используется Портным для подбора корректирующих материалов и Диагностом для определения зон развития.
