# IND.3.11.01

**Name:** Диагностическое состояние
**Name (EN):** Diagnostic state
**Type:** categorical
**Format:** enum

## Flags
- for_prompts: true
- for_qualification: true
- trainee_model: true

## Enum Values
```json
["chaos", "stuck", "turn", "development"]
```

## Description

Текущее состояние ученика в цикле развития:
- **chaos** -- нет ритма, нет целей, хаотичная активность или отсутствие активности
- **stuck** -- есть ритм, но нет прогресса (плато: те же ошибки, та же ступень)
- **turn** -- переломный момент: осознание проблемы, начало изменений
- **development** -- устойчивый рост по нескольким характеристикам

Вычисляется из: IND.3.1 (агентность) + IND.3.2 (мастерство) + IND.3.4 (ступень) + динамика за 4 недели.
