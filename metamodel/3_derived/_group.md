# 3. Расчётные показатели (IND.3.*)

**Тип:** Вторичные данные
**Источник:** Вычисляются системой, хранятся
**Права:** Пользователь — Read, Проводник — Read, Система — Write (calc)

## Описание

Расчётные показатели — это метрики, которые система вычисляет на основе первичных данных (IND.1.* и IND.2.*). Результаты вычислений сохраняются для последующего использования.

## Подгруппы

- **3_1_agency** — Агентность
- **3_2_mastery** — Мастерство
- **3_3_characteristics** — Характеристики по ступеням
- **3_4_qualification** — Квалификация и ступени
- **3_5_roles** — Мастерство по ролям
- **3_6_resourcefulness** — Ресурсность
- **3_7_community** — Сообщество и вклад
- **3_8_economics** — Экономика участия
- **3_9_ai_usage** — Использование ИИ
- **3_10_integral** — Интегральные профили и риски
