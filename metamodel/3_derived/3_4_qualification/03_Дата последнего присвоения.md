# IND.3.4.03

**Name:** Дата последнего изменения ступени
**Name (EN):** Last stage change date
**Type:** temporal
**Format:** structured_object

## Description

Дата последнего автоматического обновления ступени по каждой роли Созидателя (от профайлера R28).

Не путать с датой присвоения Степени квалификации (DEG) — та хранится в `IND.3.8` с метаданными методсовета.

## Формат

```json
{
  "Learner":      "2026-03-15",
  "Intellectual": "2026-03-01",
  "Professional": "2026-02-15",
  "Researcher":   "2026-01-20",
  "Enlightener":  "2026-01-20"
}
```
