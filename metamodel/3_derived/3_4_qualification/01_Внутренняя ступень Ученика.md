# IND.3.4.01

**Name:** Ступени ролей Созидателя
**Name (EN):** Creator role stages
**Type:** categorical
**Format:** structured_object

## Description

Автоматически вычисляемые ступени для каждой из 5 ролей Созидателя. Профайлер R28 рассчитывает ежедневно на основе поведенческих данных (agency, mastery, ритм).

Ступень — мера качества исполнения роли, не уровень доступа и не профессиональная аттестация.

## Формат

```json
{
  "Learner":      "STG.Learner.Systematic",
  "Intellectual": "STG.Intellectual.Applying",
  "Professional": "STG.Professional.Focused",
  "Researcher":   "STG.Researcher.Curious",
  "Enlightener":  "STG.Enlightener.Sharing"
}
```

## Допустимые значения

**Learner:** `STG.Learner.Random` | `STG.Learner.Practicing` | `STG.Learner.Systematic` | `STG.Learner.Disciplined` | `STG.Learner.Proactive`

**Intellectual:** `STG.Intellectual.Novice` | `STG.Intellectual.Applying` | `STG.Intellectual.Connecting` | `STG.Intellectual.Generating` | `STG.Intellectual.Teaching`

**Professional:** `STG.Professional.Reactive` | `STG.Professional.Focused` | `STG.Professional.Systematic` | `STG.Professional.Transforming` | `STG.Professional.Scaling`

**Researcher:** `STG.Researcher.Curious` | `STG.Researcher.Exploring` | `STG.Researcher.Hypothesizing` | `STG.Researcher.Contributing` | `STG.Researcher.Leading`

**Enlightener:** `STG.Enlightener.Sharing` | `STG.Enlightener.Explaining` | `STG.Enlightener.Creating` | `STG.Enlightener.Community` | `STG.Enlightener.Cultural`
