# Stages (Ступени)

Version: 0.4

Ступени — мера качества исполнения роли Созидателя. Каждая из 5 ролей имеет свои ступени. Ступень растёт через накопленный опыт и вычисляется профайлером R28 автоматически на основе поведенческих данных.

**Ступень ≠ Степень квалификации.** Ступень — автоматически измеряемое мастерство внутри конкретной роли. Степень — профессиональная аттестация методсовета МИМ, назначается вручную.

---

## Роль: Ученик (Learner)

Ступени отражают ритм и дисциплину обучения.

| Код | Название | Описание |
|-----|---------|---------|
| `STG.Learner.Random` | Случайный | Нет устойчивого учебного ритма |
| `STG.Learner.Practicing` | Практикующий | Регулярная практика, базовые требования выполняются |
| `STG.Learner.Systematic` | Систематический | Систематическое обучение с рефлексией |
| `STG.Learner.Disciplined` | Дисциплинированный | Высокая дисциплина, устойчивые привычки |
| `STG.Learner.Proactive` | Проактивный | Максимальная вовлечённость, самостоятельная инициатива |

---

## Роль: Интеллектуал (Intellectual)

Ступени отражают глубину и системность работы с знаниями.

| Код | Название | Описание |
|-----|---------|---------|
| `STG.Intellectual.Novice` | Знакомый | Знаком с концептами, но не применяет системно |
| `STG.Intellectual.Applying` | Применяющий | Регулярно использует концепты в рассуждениях |
| `STG.Intellectual.Connecting` | Связывающий | Видит связи между концептами, строит модели |
| `STG.Intellectual.Generating` | Генерирующий | Создаёт новые концепты и различения |
| `STG.Intellectual.Teaching` | Передающий | Объясняет и передаёт знания другим |

---

## Роль: Профессионал (Professional)

Ступени отражают качество работы и организации труда.

| Код | Название | Описание |
|-----|---------|---------|
| `STG.Professional.Reactive` | Реактивный | Работает по задачам без системной логики |
| `STG.Professional.Focused` | Сфокусированный | Концентрируется на ключевых объектах предметной области |
| `STG.Professional.Systematic` | Системный | Применяет метод, мыслит системами |
| `STG.Professional.Transforming` | Преобразующий | Меняет культуру работы вокруг себя |
| `STG.Professional.Scaling` | Масштабирующий | Изменяет культуру за пределами своей команды |

---

## Роль: Исследователь (Researcher)

Ступени отражают самостоятельность и глубину исследовательской деятельности.

| Код | Название | Описание |
|-----|---------|---------|
| `STG.Researcher.Curious` | Любознательный | Задаёт вопросы, ищет информацию |
| `STG.Researcher.Exploring` | Исследующий | Проводит самостоятельный анализ |
| `STG.Researcher.Hypothesizing` | Гипотезирующий | Формулирует и проверяет гипотезы |
| `STG.Researcher.Contributing` | Вносящий вклад | Создаёт оригинальные результаты |
| `STG.Researcher.Leading` | Ведущий | Организует исследования, развивает направление |

---

## Роль: Просветитель (Enlightener)

Ступени отражают масштаб и влияние передачи знания.

| Код | Название | Описание |
|-----|---------|---------|
| `STG.Enlightener.Sharing` | Делящийся | Делится знаниями с ближайшим окружением |
| `STG.Enlightener.Explaining` | Объясняющий | Системно объясняет другим, получает обратную связь |
| `STG.Enlightener.Creating` | Создающий | Создаёт учебные материалы и форматы |
| `STG.Enlightener.Community` | Сообщественный | Формирует сообщество вокруг идей |
| `STG.Enlightener.Cultural` | Культурный | Вносит вклад в культуру отрасли или общества |

---

## Примечание

Ступени для ролей Интеллектуал, Профессионал, Исследователь, Просветитель — черновик (v0.4), требуют валидации на реальных данных. Ступени роли Ученик — верифицированы в продакшне.
