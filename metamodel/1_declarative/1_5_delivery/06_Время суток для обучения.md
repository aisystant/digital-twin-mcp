# IND.1.5.06

**Name:** Время суток для обучения
**Name (EN):** Preferred study time
**Type:** categorical
**Format:** enum

## Flags
- for_prompts: true
- for_qualification: false
- trainee_model: true

## Enum Values
```json
["morning", "afternoon", "evening"]
```

## Description

Предпочитаемое время суток для учебных сессий:
- **morning** -- утро (до 12:00)
- **afternoon** -- день (12:00--18:00)
- **evening** -- вечер (после 18:00)

Источник: онбординг (schedule_time). Используется для планирования напоминаний и доставки материалов.
