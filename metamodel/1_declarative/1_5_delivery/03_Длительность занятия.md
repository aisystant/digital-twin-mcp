# IND.1.5.03

**Name:** Длительность занятия
**Name (EN):** Session duration
**Type:** temporal
**Format:** integer
**Unit:** minutes

## Flags
- for_prompts: true
- for_qualification: false
- trainee_model: true

## Description

Предпочитаемая длительность одного учебного занятия в минутах.
Типичные значения: 15, 30, 60.

Источник: онбординг (study_duration). Объём порции материала вычисляется автоматически.
