# IND.1.5.01

**Name:** Формат подачи
**Name (EN):** Delivery format
**Type:** categorical
**Format:** enum

## Flags
- for_prompts: true
- for_qualification: false
- trainee_model: true

## Enum Values
```json
["examples", "tasks", "theory", "mix"]
```

## Description

Предпочитаемый формат учебного материала:
- **examples** -- через разбор примеров и кейсов
- **tasks** -- через практические задания
- **theory** -- через объяснение концепций и принципов
- **mix** -- смешанный формат (по умолчанию)

Источник: онбординг или настройки профиля. Адаптируется по feedback.
