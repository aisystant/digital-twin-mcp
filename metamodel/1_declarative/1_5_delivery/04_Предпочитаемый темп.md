# IND.1.5.04

**Name:** Предпочитаемый темп
**Name (EN):** Preferred pace
**Type:** categorical
**Format:** enum

## Flags
- for_prompts: true
- for_qualification: false
- trainee_model: true

## Enum Values
```json
["slow", "normal", "intensive"]
```

## Description

Предпочитаемая интенсивность подачи нового материала:
- **slow** -- меньше нового за сессию, больше повторения и закрепления
- **normal** -- сбалансированный темп (по умолчанию)
- **intensive** -- максимум нового, минимум повторения

Начальное значение: декларируется пользователем. В будущем адаптируется по данным: если completion_rate падает при intensive -- система рекомендует снизить темп.
