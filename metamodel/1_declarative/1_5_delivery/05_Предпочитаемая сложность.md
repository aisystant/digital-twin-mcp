# IND.1.5.05

**Name:** Предпочитаемая сложность
**Name (EN):** Preferred difficulty
**Type:** categorical
**Format:** enum

## Flags
- for_prompts: true
- for_qualification: false
- trainee_model: true

## Enum Values
```json
["basic", "advanced", "adaptive"]
```

## Description

Предпочитаемый уровень сложности учебного материала:
- **basic** -- упрощённые объяснения, больше поддержки
- **advanced** -- продвинутый уровень, меньше scaffolding
- **adaptive** -- система подбирает автоматически по данным (по умолчанию)

При adaptive: сложность определяется из IND.3.4.01 (ступень ученика) и IND.3.11.01 (когнитивная нагрузка).
