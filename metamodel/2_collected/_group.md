# 2. Собираемые показатели (IND.2.*)

**Тип:** Первичные данные
**Источник:** Автоматически из действий пользователя
**Права:** Пользователь — Read, Проводник — Read, Система — Write

## Описание

Собираемые показатели — это данные, которые система получает автоматически из действий пользователя: прохождение курсов, время сессий, платежи, активность в клубе, логи взаимодействия с ИИ.

## Подгруппы

- **2_1_account** — Аккаунт и сессии
- **2_2_courses** — Прохождение курсов
- **2_3_practice** — Задания и практики
- **2_4_time** — Время и ритм
- **2_5_community** — Сообщество (Клуб)
- **2_6_coding** — Рабочее время и кодирование (WakaTime)
- **2_7_iwe** — Платформенная активность (IWE git, Claude Code, РП)
- **2_8_ai_logs** — Логи ботов и ИИ
- **2_9_finance** — Финансы
