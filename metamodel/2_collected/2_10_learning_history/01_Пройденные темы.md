# IND.2.10.01

**Name:** Пройденные темы
**Name (EN):** Completed topics
**Type:** temporal
**Format:** structured_text

## Flags
- for_prompts: false
- for_qualification: true
- trainee_model: false

## Description

Хронологический список пройденных тем с метаданными. Структура:

```json
[
  {
    "topic_id": "string",
    "topic_title": "string",
    "source": "marathon|feed|tailor",
    "completed_at": "ISO date",
    "complexity_level": 1-5,
    "fixation_length": 0
  }
]
```

Источник: события `marathon_step`, `feed_completed`, `learning_completed` из Event Store.
Используется для: построения learning path, выявления пробелов, рекомендации следующей темы.
