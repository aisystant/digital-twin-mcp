# IND.2.10.02

**Name:** Результаты тренировок
**Name (EN):** Training results
**Type:** temporal
**Format:** structured_text

## Flags
- for_prompts: false
- for_qualification: true
- trainee_model: false

## Description

История попыток тренировок принципов. Структура:

```json
[
  {
    "principle_id": "string",
    "depth": 1-3,
    "passed": true,
    "attempt_at": "ISO date",
    "p_mastery": 0.0-1.0
  }
]
```

Источник: события `training_attempt` из Event Store. Поле `p_mastery` вычисляется BKT (IND.3.2.04).
Используется для: Knowledge Tracing, определения зон мастерства и пробелов, входные данные для IND.3.11.03 (рекомендуемый фокус).
